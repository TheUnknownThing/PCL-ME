# Guide for Agents

This project ports the well-known Minecraft Launcher [PCL2-CE](https://github.com/PCL-Community/PCL2-CE) to all platforms including Windows, Linux and macOS. It uses Avalonia as the UI framework and .NET 10 as the runtime.

If `/PCL.Frontend.WPF` is present in the codebase, it contains the old WPF-based frontend, which is no longer maintained in this project. Agents asked either to review or build the project should instead focus on the `/PCL.Frontend.Avalonia` directory.
