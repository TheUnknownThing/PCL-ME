## 貢獻說明

- 新的介面改動應優先提交到 `PCL.Frontend.Avalonia/`。
- 目前主線技術棧是 `C# + .NET 10 + Avalonia`。
- 本倉庫不再以舊版 VB.NET / WPF 前端作為維護方向。
- Pull Request 若附帶跨平台驗證資訊會很有幫助。

## 回報問題時建議附帶

1. 清楚的重現步驟。
2. 實際結果與預期結果。
3. 你的作業系統、架構與 .NET 環境資訊。
4. 相關日誌、截圖或診斷檔案。
