## Contribution Notes

- New UI work should target `PCL.Frontend.Avalonia/`.
- The project uses `C# + .NET 10 + Avalonia`.
- Cross-platform validation details are especially helpful in pull requests.

## Reporting Problems

Please include:

1. Clear reproduction steps.
2. The actual result and the expected result.
3. Your operating system, architecture, and .NET environment.
4. Logs, screenshots, or diagnostics when available.
