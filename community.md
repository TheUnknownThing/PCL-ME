## 参与方式

- [提交 Issue](https://github.com/TheUnknownThing/PCL-CE/issues)
- [贡献指南](https://github.com/TheUnknownThing/PCL-CE/blob/dev/CONTRIBUTING.md)
- [仓库主页](https://github.com/TheUnknownThing/PCL-CE)

## 贡献说明

- 新的界面改动应优先提交到 `PCL.Frontend.Avalonia/`。
- 当前主线技术栈是 `C# + .NET 10 + Avalonia`。
- Pull Request 中附带跨平台验证信息会很有帮助。

## 反馈问题时建议附带

1. 清晰的复现步骤。
2. 实际结果与预期结果。
3. 你的操作系统、架构与 .NET 环境信息。
4. 相关日志、截图或诊断文件。
